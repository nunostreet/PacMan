### Config file structure

```json
{
  "highscore_filename": "highscores.json",
  "lives": 3,
  "seed": 42,
  "level_max_time": 90,
  "pacgum_count": 42,
  "points_per_pacgum": 10,
  "points_per_super_pacgum": 50,
  "points_per_ghost": 200,
  "ghost_respawn_time": 5.0,
  "levels": [
    { "width": 15, "height": 11 },
    ...
  ]
}
```
